Como Executar o Software

  1. Baixe o arquivo na seção de download abaixo.
  2. Extraia o conteúdo do arquivo ZIP em uma pasta de sua preferência.
  3. Abra o terminal ou prompt de comando.
  4. Navegue até a pasta extraída.
  5. Execute o comando python app.py (é necessário ter o Python instalado).
  6. Acesse o software através do navegador em http://localhost:5000.
